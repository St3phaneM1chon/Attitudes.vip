3. Authenticate with required services
4. Start using natural language to interact

### Development
- SDKs available: Python, TypeScript, C#, Java
- Protocol: JSON-RPC 2.0
- Transport: stdio, HTTP/SSE
- Authentication: OAuth 2.1 standard
